Key Features

Media Management

Automated media collection, organization, and transcoding with popular tools like Radarr, Sonarr, and Tdarr.

Server Administration

Comprehensive server management with Portainer, Watchtower, and Tautulli for monitoring and maintenance.

Secure & Automated

Built on Docker and Proxmox VE with automated updates and secure remote access through Cloudflare tunnels.
